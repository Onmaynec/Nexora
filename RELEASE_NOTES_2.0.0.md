# Nexora 2.0.0 — Violet Grid Stable

Дата подготовки: 20 июля 2026.

## Главное

2.0.0 объединяет ранее запланированные обновления данных, аккаунтов, комнат, файлов, безопасности и интерфейса. Добавлены Nexora Plus/Pulse и GitHub auto-update Client. Звонки и E2EE исключены из объёма релиза по требованию владельца.

## Исправления по тестовым скриншотам

- клик по аватару в сообщении, списке чатов, контактах, поиске, заголовке и участниках открывает полноценный профиль;
- счётчики с нулевым значением больше не отображаются;
- меню действий сообщения позиционируется внутрь ленты и не уходит под details drawer;
- picker реакций сохраняет активную hover-зону, не закрывается при переводе мыши и закрывается по выбору/Escape/клику снаружи;
- раскрывающийся dock ограничен шириной левой панели и сжимает соседние кнопки;
- меню чата перемещено в верхнюю безопасную область карточки;
- лента сообщений и вложения ограничены собственным контейнером.

## Новые механики общения

- Telegram-подобная карточка профиля с действиями, общими комнатами и статусом связи;
- «Сохранённые сообщения» как личное пространство пользователя;
- глобальный FTS5-поиск и отдельная лента сохранённых сообщений;
- закрепление, mute, архив и фильтры чатов;
- закладки, пересылка, multi-select и копирование сообщений;
- черновики на каждый чат, разделитель непрочитанных и кнопка перехода вниз;
- идемпотентная outbox-очередь и повтор неудачной отправки;
- премиальные рамки/акценты и расширенные реакции под entitlement Plus.

## Nexora Plus / Pulse

- Free-общение не ограничивается подпиской;
- Plus entitlement, 400 импульсов за расчётный месяц и скрываемый badge;
- каталог коллективных целей комнат без звонков;
- целочисленный баланс, идемпотентные вклады, защита от overfund и room entitlement после финансирования;
- автоматический возврат sandbox-взносов при отмене/истечении цели;
- отдельный безопасный Sandbox;
- production checkout только через HTTPS Pulse Cloud;
- Ed25519-проверка подписанных ответов, scope/expiry validation и локальный cache.

Реальные платежи не включаются без production Pulse Cloud и провайдера оплаты.

## Хранилище и надёжность

- SQLite schema 5 и автоматическая миграция существующего `nexora.json`/старых схем;
- отдельные таблицы пользователей, сессий, комнат, сообщений, файлов, закладок, uploads и Pulse;
- diff/UPSERT persistence в одной `BEGIN IMMEDIATE` транзакции;
- WAL + `synchronous=FULL`, integrity check на старте и crash recovery;
- автоматические/ручные/зашифрованные копии, restore со страховочной копией;
- квота, статистика, retention, экспорт комнаты и очистка потерянных файлов.

## Windows и обновления

- Client автоматически получает стабильные обновления из GitHub Releases `Onmaynec/Nexora`;
- Server остаётся управляемым обновлением администратора;
- CI проверяет Windows и Linux;
- тег `v2.0.0` запускает подписанную Windows-сборку и формирует SHA-256 checksums.
