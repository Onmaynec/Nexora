# Nexora 1.0.1 — Windows Stable Candidate

## Главное исправление 1.0.1

Удалена нативная зависимость `better-sqlite3`. Сервер использует `node:sqlite`, поэтому `npm run dist:windows` не вызывает `node-gyp`, не ищет Visual Studio и не зависит от пробелов/кириллицы в пути проекта. Минимальная версия Node для сборки — 22.16; рекомендуется Node 24 LTS.

## Данные

- SQLite schema 4 и миграции JSON/schema 3;
- WAL, `synchronous=FULL`, транзакции и integrity check;
- автоматические и ручные копии;
- опциональное AES-256-GCM шифрование копии;
- квота, статистика, retention, очистка сирот и экспорт комнаты.

## Комнаты и аккаунты

- аватары, статусы, смена/сброс пароля, активные сессии;
- модераторы, передача владения, удаление/бан;
- read-only, slow mode, ограничения файлов/голосовых;
- заявки, ограниченные приглашения и аудит действий;
- системные сообщения вступления/выхода.

## Файлы и голосовые

- drag-and-drop, множественная отправка, прогресс, отмена и повтор;
- предварительная проверка места, миниатюры, документы и медиа чата;
- waveform, pause/resume, предпрослушивание, noise gate;
- скорости 1×/1.5×/2×, глобальное продолжение и признак прослушивания.

## Безопасность и релиз

- Origin + CSRF, постоянные rate limits, login audit и временный lock;
- настраиваемая политика паролей и обязательная смена временного;
- Server ID, список серверов и certificate pinning;
- совместимость Client/Server и каркас обновлений;
- проверка release signing и журналы ошибок.

## Проверено в этой поставке

- `npm run check`;
- 29/29 автоматических тестов;
- load test 20 клиентов / 120 сообщений;
- аварийное завершение записи SQLite;
- миграция schema 3 → 4;
- шифрование/проверка резервной копии;
- production dependency audit: 0 high / 0 critical;
- короткий soak smoke: integrity `ok`.

## Требует внешнего Windows-подтверждения

В исходную поставку не может быть включена реальная подпись без закрытого сертификата владельца. До маркировки публичного релиза как Stable нужно на Windows 10 и 11 выполнить `npm run release:windows`, проверить подпись `.exe`, чистую установку, обновление со старой версии и длительный soak (рекомендуется 24 часа).

Звонки и E2EE не входят в 1.0.1.
