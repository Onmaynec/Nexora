# Nexora Release History

Хронологическая история версий поддерживается **только** в [CHANGELOG.md](CHANGELOG.md). Этот файл намеренно не дублирует список выпусков, чтобы репозиторий, документация и GitHub Release не расходились.

Для текущего выпуска используйте:

- [Release Notes 3.3.2](RELEASE_NOTES_3.3.2.md) — scope и пользовательское влияние;
- [Release Verification 3.3.2](RELEASE_VERIFICATION_3.3.2.md) — acceptance contract, CI и asset smoke requirements;
- [Current release evidence](release-evidence/current.json) — машиночитаемая версия и tag;
- `release-evidence/v3.3.2.json` — неизменяемое свидетельство опубликованных assets после выпуска.

Release-specific notes и verification documents являются историческими свидетельствами и не заменяют каноническую временную шкалу в `CHANGELOG.md`.
